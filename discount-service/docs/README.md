# Discount Service - Сервис поиска и сбора скидок

## Описание
Сервис по поиску и сбору скидок, промо-акций, купонов на товары и услуги с монетизацией через подписку.

## Архитектура проекта

```
discount-service/
├── backend/              # Основной API сервис
│   ├── app/             # Основное приложение FastAPI
│   ├── models/          # Модели данных
│   ├── services/        # Бизнес-логика
│   ├── api/            # API endpoints
│   └── config/         # Конфигурация
├── scraper/            # Сервис автоматического поиска скидок
│   ├── spiders/        # Парсеры для разных источников
│   ├── processors/     # Обработка найденных данных
│   └── schedulers/     # Планировщик задач
├── frontend/           # Web интерфейс
└── docs/              # Документация
```

## Функционал

### Основной сервис (backend)
1. **Поиск и фильтрация** скидок по категориям товаров и услуг
2. **Персональные скидки** на основе истории покупок и предпочтений
3. **Семейная карта** - общие скидки для семьи
4. **Базовые скидки** (2-3%) на товары без специальных предложений
5. **Цифровые каталоги** магазинов (Перекресток, Пятерочка и др.)
6. **Подборки предложений** - создание и публикация своих коллекций
7. **Банковские категории** - специальные скидки раз в месяц

### Скрапер сервис (scraper)
1. **Автоматический поиск** источников скидок в интернете
2. **Парсинг сайтов** магазинов и агрегаторов
3. **Регулярное обновление** базы данных
4. **Валидация и фильтрация** найденных предложений
5. **Добавление в базу** с проверкой дубликатов

## Технологии

### Backend
- **FastAPI** - современный async веб-фреймворк
- **SQLAlchemy** - ORM для работы с БД
- **PostgreSQL** - основная база данных
- **Redis** - кэширование и очереди задач
- **Celery** - фоновые задачи
- **Pydantic** - валидация данных

### Scraper
- **Scrapy** - фреймворк для веб-скрапинга
- **Selenium** - для JavaScript-heavy сайтов
- **BeautifulSoup4** - парсинг HTML
- **Playwright** - автоматизация браузеров

### Frontend
- **React** или **Vue.js** - современный UI
- **TypeScript** - типизация
- **TailwindCSS** - стилизация

### Инфраструктура
- **Docker** - контейнеризация
- **Docker Compose** - оркестрация
- **Nginx** - reverse proxy
- **Prometheus + Grafana** - мониторинг

## Модель данных

### Основные сущности:
- **User** - пользователи с подписками
- **Discount** - скидки и предложения
- **Category** - категории товаров/услуг
- **Store** - магазины и сети
- **Collection** - пользовательские подборки
- **FamilyCard** - семейные карты
- **Subscription** - подписки пользователей

## API Endpoints

### Аутентификация
- POST /api/auth/register
- POST /api/auth/login
- POST /api/auth/logout

### Пользователи
- GET /api/users/profile
- PUT /api/users/profile
- GET /api/users/subscriptions

### Скидки
- GET /api/discounts - список с фильтрацией
- GET /api/discounts/{id} - детальная информация
- GET /api/discounts/personal - персональные скидки
- GET /api/discounts/stores/{store_id} - скидки магазина

### Категории
- GET /api/categories - все категории
- GET /api/categories/{id}/discounts - скидки категории

### Подборки
- GET /api/collections - мои подборки
- POST /api/collections - создать подборку
- PUT /api/collections/{id} - обновить
- DELETE /api/collections/{id} - удалить
- POST /api/collections/{id}/share - поделиться

### Магазины
- GET /api/stores - список магазинов
- GET /api/stores/{id} - информация о магазине
- GET /api/stores/{id}/catalog - цифровой каталог

### Семейная карта
- GET /api/family/cards - мои карты
- POST /api/family/cards - создать карту
- POST /api/family/cards/{id}/members - добавить участника

## Запуск проекта

```bash
# Установка зависимостей
docker-compose up -d

# Миграции БД
docker-compose exec backend alembic upgrade head

# Запуск скрапера
docker-compose exec scraper scrapy crawl all_stores

# Просмотр логов
docker-compose logs -f
```
